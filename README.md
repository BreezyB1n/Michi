# Michi

Michi is a web UI prototype for a Guide Agent that helps users turn a natural-language goal into an understandable, in-page guide through a complex product.

The first milestone is a local web demo of the Cloudflare Workers primary guide path. Product and implementation work should start from the specs in `docs/product-specs/`.

## Local development

```bash
npm install
npm run dev
```

Useful checks:

```bash
npm test
npm run build
npm run test:e2e
```
